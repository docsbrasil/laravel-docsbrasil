# Compilação de Assets (Vite)

## Introdução

<a href="https://vitejs.dev" target="_blank">Vite</a> é uma ferramenta moderna de compilação que fornece um ambiente de desenvolvimento extremamente rápido e empacota seu código para produção. Ao construir aplicações com Laravel, você normalmente usará o Vite para empacotar os arquivos CSS e JavaScript da sua aplicação em ativos prontos para produção.

O Laravel se integra perfeitamente com o Vite, fornecendo um plugin oficial e uma diretiva Blade para carregar seus ativos para desenvolvimento e produção.

::: tip
"Empacotar" é o processo de combinar vários arquivos em um único arquivo. Isso é feito para reduzir o número de solicitações de rede necessárias para carregar uma página da web.
:::
